# AI Movie Studio

منصة لإنشاء فيلم قصير من فكرة واحدة: تخطيط القصة، مشاهد وشخصيات ثابتة، صور، تعليق صوتي، ثم فيديو MP4.

## التشغيل المحلي

1. انسخ `backend/.env.example` إلى `backend/.env` واضبط المزودات التي تريد استعمالها.
2. من مجلد `backend`: أنشئ بيئة Python وثبّت `pip install -r requirements.txt`، ثم شغّل `uvicorn app.main:app --reload --port 8000`.
3. من مجلد `frontend`: نفّذ `npm install` ثم `npm run dev`.

افتراضيًا يستخدم التطبيق OpenAI (إن ضبطت `OPENAI_API_KEY`) أو Ollama محليًا لكتابة السيناريو. **توليد الصور حاليًا هو صور معاينة (preview) مُولَّدة برمجيًا وليس نموذج ذكاء اصطناعي حقيقي (لا يوجد تكامل Stable Diffusion في الكود)** — الكود يحتوي على نقطة ربط (`image_for_scene`) جاهزة لإضافة مزوّد صور لاحقًا. عند عدم توفر أي مزوّد للسيناريو، يعرض النظام قصة احتياطية (fallback) بدل الفشل.

## ما يشمله الإصدار فعليًا

- مشاريع محفوظة، حالة مهمة وتقدم قابل للاستئناف، وسجل أخطاء واضح.
- سيناريو ومشاهد منظمة وشخصيات مع بطاقة مرجعية مشتركة في كل مطالبة صور.
- فحص مخرجات الصور والصوت قبل تجميع الفيديو (بدون إعادة محاولة تلقائية داخل خط الإنتاج؛ يوجد endpoint `/retry` يدوي).
- تعليق صوتي واحد متصل للفيلم وتصدير H.264/AAC.
- واجهة عربية متجاوبة لاختيار النوع والأسلوب والمدة ومتابعة الإنتاج.
- المعالجة الخلفية تعمل عبر FastAPI BackgroundTasks (وليس Redis/Celery). هذا كافٍ لعدد صغير من المستخدمين المتزامنين؛ **Redis/Celery وStable Diffusion الحقيقي هما ترقيات مستقبلية غير مطبّقة بعد**، مذكورة في `DEPLOY.md` كخطوة لاحقة عند نمو الاستخدام، وليستا جزءًا من هذا الإصدار.
